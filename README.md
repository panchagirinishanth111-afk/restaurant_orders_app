# Restaurant Order Board
This ZIP contains everything needed to run your restaurant order board on GitHub Pages.

## How to deploy on GitHub Pages:
1. Create a new GitHub repository.
2. Upload the contents of this ZIP (index.html and README.md).
3. Go to Settings â†’ Pages.
4. Select:
   - Branch: main
   - Folder: / (root)
5. Save.
6. Your page will be available at:
   https://your-username.github.io/your-repository-name/

## IMPORTANT:
Replace YOUR_KEY, YOUR_DOMAIN, YOUR_PROJECT inside index.html with your Firebase project config.
